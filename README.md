# 🥷 Cadastro de Ninjas

Projeto desenvolvido para praticar **Java** e **Programação Orientada a
Objetos (POO)**.\
O sistema simula um pequeno cadastro de ninjas, permitindo organizar
informações e treinar conceitos fundamentais da linguagem.

------------------------------------------------------------------------

## 📌 Sobre o Projeto

O **CadastroDeNinjas** foi criado durante estudos de Java para
exercitar:

-   Criação de classes
-   Encapsulamento
-   Métodos
-   Sobrescrita de métodos (`@Override`)
-   Organização de código
-   Uso de Git e GitHub

------------------------------------------------------------------------

## 🚀 Como clonar o projeto

``` bash
git clone https://github.com/amarildorpg/CadastroDeNinjas.git
```

Entrar na pasta:

``` bash
cd CadastroDeNinjas
```

------------------------------------------------------------------------

## 🔗 Configurar repositório remoto

Se precisar alterar o repositório remoto:

``` bash
git remote set-url origin https://github.com/amarildorpg/CadastroDeNinjas.git
```

Verificar o remoto configurado:

``` bash
git remote -v
```
